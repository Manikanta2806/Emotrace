System architecture

Two independent, comparable pipelines — Base and Enhanced — sharing a comparison and metrics layer.

Base Model Pipeline
1

Input Processing

Accepts RECCON-format dicts, raw text, or utterance lists. Auto-detects "Speaker: text" dialogue format vs plain sentences.

2

Emotion Detection

j-hartmann/emotion-english-distilroberta-base (HuggingFace). Returns all 7 emotion scores — takes the highest-confidence non-neutral label.

3

Utterance Pair Generation

Generates all candidate (cause → emotion) pairs. Temporal constraint enforced: cause_index ≤ emotion_index.

4

Implicit Cause Generation (Flan-T5)

google/flan-t5-large generates 4 distinct causal hypotheses per pair using 4 orthogonal prompt angles: triggering event, violated belief, personal stakes, and social dynamic. Near-duplicates are removed automatically.

5

Semantic Scoring (SBERT)

all-MiniLM-L6-v2 scores each IC against the emotion utterance (weight 0.6) and the cause utterance (weight 0.4). Best IC = highest combined score.

6

Causal Reasoning

Flan-T5 yes/no judgment + SBERT threshold (≥0.20) + temporal check → is_causal flag. Produces clean natural-language explanation.

Enhanced Model Pipeline
1–5

Same as Base

Input processing, emotion detection, pair generation, IC generation, and semantic scoring are identical.

6

Implicit Cause Quality Analysis NEW

Each IC scored on 3 model-based dimensions:
Semantic Relevance (0.40 weight) — SBERT cosine similarity to emotion utterance
Specificity (0.35 weight) — Flan-T5 yes/no judgment: "is this specific and concrete?"
Sentiment Alignment (0.25 weight) — cardiffnlp/twitter-roberta-base-sentiment-latest polarity vs expected emotion
Quality ≥ 0.42 → Strong; below → Weak

7

Quality Filtering + Enhanced Reasoning NEW

Only Strong ICs are used for final reasoning. Falls back to best Weak IC with score penalty if none qualify. Score formula: 0.30×direct + 0.25×cause_impl + 0.20×quality + 0.05×semantic + LLM_bonus.

Comparison & API Layer
API

FastAPI Endpoints

POST /analyze/base · POST /analyze/enhanced · POST /compare
Each returns: emotion, cause, explanation, causal score, ranked ICs with quality dimensions.

CMP

Comparison System

Runs both pipelines independently. Computes Accuracy, Precision, Recall, F1 — against ground truth if provided, heuristic otherwise. Qualitative agreement analysis + error case detection. Declares winner by ΔF1 > 0.01.
